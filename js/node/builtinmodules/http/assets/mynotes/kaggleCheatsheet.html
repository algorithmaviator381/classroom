Kaggle competitions cheat sheet

You could throw pretty much the same script at most competitions and achieve a great score. The script goes like this:

  • Import data
  • Replace missing values
  • Encode categorical variables
  • Do some one hot encoding
  • Run XGB, RF, NN
  • Ensemble XGB1, RF1, NN1 using both XGB2 and NN2 as metamodel
  • Use XGB3 and NN3 as second layer metamodels to ensemble XGB2 and NN2
